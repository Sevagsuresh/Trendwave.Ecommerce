🛍️ Trendware – E-Commerce Web Application

Trendware is a modern full-stack e-commerce web application designed to provide a smooth online shopping experience with secure authentication, product management, and order handling.
Built using the MERN stack, it focuses on performance, scalability, and clean UI.

🚀 Features
👤 User Features

User registration & login (JWT authentication)

Browse products by category

Product search & filtering

Add to cart & manage cart items

Secure checkout process

Order history tracking

🛒 Admin Features

Admin dashboard

Add, update & delete products

Manage users

View and manage orders

Product stock management

🧰 Tech Stack

Frontend

React.js

Redux / Context API

HTML5, CSS3

Axios

Backend

Node.js

Express.js

JWT Authentication

Database

MongoDB (Mongoose)

Other Tools

Git & GitHub

Postman

REST APIs
